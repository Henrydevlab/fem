# 🎯 Frontend Mentor Solutions

A central index for my [Frontend Mentor](https://www.frontendmentor.io) project solutions. This dashboard tracks my completed builds, code repositories, and live previews in one place.

## 🛠️ Build Standards

* **Semantic HTML5:** Clean document outlines and accessible elements.
* **A11y & SEO:** Image lazy-loading, explicit `alt` text, and proper ARIA attributes for high Lighthouse scores.
* **Responsive Layouts:** CSS Grid, Flexbox, and fluid typography for mobile, tablet, and desktop viewports.
* **Conventional Commits:** Structured git workflows (`feat:`, `style:`, `docs:`) for clear history.
